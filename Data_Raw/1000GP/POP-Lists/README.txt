This folder should only contain 1000GP (phase 3) population lists, in the following format (available at https://www.internationalgenome.org/data-portal/population):

1. phase3_POP.txt

FORMAT:	simply change first column header, to #IID, and should all work fine
	------------------------------------------
	#IID	Sex	etc...
	ABXXXXX	m/f	___...
	------------------------------------------

where	POP	is the population reference code 	e.g. 'GBR'
	ABXXXXX	is the sample reference code 	e.g. 'HG00106'
